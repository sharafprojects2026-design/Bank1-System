# Bank Management System - C++

A simple Bank Management System developed using C++.

## Features
- Show all clients
- Add new client
- Delete client
- Update client information
- Find client by account number
- Save data to file

## Technologies Used
- C++
- File Handling
- Structures
- Vectors
- Functions

## Project Description
This project is a console-based banking system that allows managing clients information using file storage.

The system stores:
- Account Number
- Pin Code
- Client Name
- Phone Number
- Account Balance

All data is saved inside a text file.
